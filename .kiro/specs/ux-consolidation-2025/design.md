## Visual Identity

### Design Philosophy

The consolidation maintains AuraStream's existing visual language while improving information hierarchy and reducing cognitive load. No new design patterns are introduced - we're reorganizing existing patterns for clarity.

**Core Principles:**
1. **Consistency** - Use existing components and patterns
2. **Clarity** - Clear visual hierarchy in navigation
3. **Continuity** - Smooth transitions between old and new
4. **Confidence** - Users know where they are at all times

## Accessibility

### Keyboard Navigation

- `Tab` - Move between tabs
- `Arrow Left/Right` - Switch tabs
- `Enter/Space` - Activate tab
- `Escape` - Close toast

### Screen Reader

- Tabs: `role="tablist"`, `role="tab"`, `role="tabpanel"`
- Active tab: `aria-selected="true"`
- Tab panel: `aria-labelledby` linked to tab
- Toast: `role="alert"`, `aria-live="polite"`

### Focus Management

- Tab focus visible with ring
- Tab panel receives focus on switch
- Toast dismissible via keyboard
