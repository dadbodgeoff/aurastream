# Aurastream Mobile App

Expo React Native application for Aurastream.

## Setup

This app will be configured in Task 6.3 with:
- Expo SDK
- TypeScript
- React Native Paper or NativeWind for styling
- TanStack Query for data fetching
- Zustand for state management

## Development

```bash
# From monorepo root
npm run dev --filter=@aurastream/mobile
```
