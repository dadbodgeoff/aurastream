# AuraStream Keyboard Shortcuts

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| ⌘K / Ctrl+K | Command Palette | Open the command palette for quick navigation and actions |
| N | New Asset | Start creating a new asset |
| B | Brand Kits | Navigate to Brand Kits page |
| A | Assets | Navigate to Assets page |
| ? | Show Shortcuts | Display this shortcuts modal |
| Escape | Close | Close any open modal or panel |

## Command Palette

When the command palette is open:
- Type to search commands
- Use ↑↓ arrows to navigate
- Press Enter to execute
- Press Escape to close

## Navigation

| Shortcut | Destination |
|----------|-------------|
| G then D | Dashboard |
| G then A | Assets |
| G then B | Brand Kits |
| G then S | Settings |

## Tips

- Shortcuts work from anywhere in the app (except when typing in inputs)
- Use the command palette (⌘K) for the fastest navigation
- Reduced motion users: animations are automatically disabled
