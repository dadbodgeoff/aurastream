# Aurastream Web App

Next.js 14 web application for Aurastream.

## Setup

This app will be configured in Task 6.2 with:
- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- TanStack Query for data fetching
- Zustand for state management

## Development

```bash
# From monorepo root
npm run dev --filter=@aurastream/web
```
